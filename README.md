# dotfiles

## How to install

### Requirements
- Bash
- RubyGems

### Deploy
- $ bash -c "$(curl -L https://raw.githubusercontent.com/applejxd/dotfiles/master/deploy.sh)"

### Initialization
- $ bash -c "$(curl -L https://raw.githubusercontent.com/applejxd/dotfiles/master/init.sh)"

## How to uninstall
- $ homesick unlink dotfiles

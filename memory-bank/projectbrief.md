# プロジェクト概要

## dotfilesプロジェクト

このプロジェクトは、複数のプラットフォーム（Linux、macOS、WSL）で一貫した開発環境を維持するための設定ファイル（dotfiles）を管理するリポジトリです。

## 目的

- 異なるマシン間で一貫した開発環境を維持する
- 新しいマシンのセットアップ時間を短縮する
- 設定ファイルを一元管理し、バージョン管理する
- 環境構築プロセスを自動化する

## 主要コンポーネント

- シェル設定（.bashrc, .zshrc）
- エディタ設定（.vimrc, .spacemacs）
- ターミナル設定（.tmux.conf, .hyper.js）
- システム設定（sshd_config, wsl.conf）
- インストールスクリプト（各種環境向け）

## 対象プラットフォーム

- Ubuntu/Debian系Linux
- macOS
- Windows Subsystem for Linux (WSL)

## 実装方針

- シンプルで理解しやすい構造を維持する
- プラットフォーム固有の設定を明確に分離する
- 再利用可能なスクリプトを作成する
- 必要に応じて段階的にカスタマイズ可能にする
